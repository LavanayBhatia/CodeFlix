How do I reverse a string in C without using strrev()?
Here's how you can reverse a string in C without using strrev(): 
void reverseString(char* str) {
    int length = strlen(str);
    int i, j;
    char temp;
    
    for (i = 0, j = length - 1; i < j; i++, j--) {
        temp = str[i];
        str[i] = str[j];
        str[j] = temp;
    }
}
This function swaps characters from both ends moving toward the center until the entire string is reversed.
